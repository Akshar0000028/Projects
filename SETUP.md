# Quick Setup Guide for Seen Plays Website

## ⚠️ IMPORTANT: Install Node.js First

Your system needs **Node.js** installed to run this React project.

### Step 1: Install Node.js

1. Go to: **https://nodejs.org/**
2. Download the **LTS version** (recommended)
3. Run the installer
4. ✅ **IMPORTANT**: Make sure to check "Add to PATH" during installation
5. Restart your computer (or at least restart VS Code/PowerShell)

### Step 2: Verify Installation

Open PowerShell and run:
```powershell
node --version
npm --version
```

You should see version numbers like:
```
v20.x.x
10.x.x
```

### Step 3: Run the Project

Once Node.js is installed, in PowerShell (in this folder), run:

```powershell
# Install dependencies (first time only)
npm install

# Start the development server
npm run dev
```

Then open **http://localhost:5173** in your browser!
