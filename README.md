# Seen Plays - Luxury Streetwear E-Commerce Website

A premium, fully functional clothing brand website built with React, Tailwind CSS, and Framer Motion.

## Features

- 🎨 **Modern Design**: Elegant black, white, beige, and gold color palette
- ✨ **Smooth Animations**: Framer Motion animations throughout
- 📱 **Fully Responsive**: Works on desktop, tablet, and mobile
- 🛍️ **E-Commerce Ready**: Product catalog, filtering, cart functionality
- 🎯 **5 Complete Pages**: Home, Shop, Product Details, About, Contact
- 🎭 **Interactive UI**: Hover effects, transitions, and parallax scrolling

## Tech Stack

- React 18
- Vite
- React Router DOM
- Tailwind CSS
- Framer Motion

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Project Structure

```
src/
├── components/     # Reusable components (Navbar, Footer, ProductCard)
├── pages/         # Page components (Home, Shop, About, Contact, ProductDetails)
├── context/       # React Context (CartContext)
├── data/          # Data files (products.js)
├── App.jsx        # Main app component with routing
├── main.jsx       # Entry point
└── index.css      # Global styles
```

## Pages

1. **Home**: Hero section, featured products, new arrivals, trending items
2. **Shop**: Product grid with category filtering
3. **Product Details**: Individual product page with size selector and add to cart
4. **About**: Brand story, mission, and founder section
5. **Contact**: Contact form and information

## Features

- Sticky navbar with cart icon
- Mobile-responsive hamburger menu
- Product filtering by category
- Shopping cart functionality
- Smooth page transitions
- Scroll-triggered animations
- Modern typography (Montserrat & Poppins)

## Customization

- Product data can be modified in `src/data/products.js`
- Colors can be customized in `tailwind.config.js`
- Fonts can be changed in `index.html` and `tailwind.config.js`
